Ingénierie

L'alliance de la Performance C et de la Structure SQL

Contrairement aux fichiers texte classiques, notre architecture utilise des structures C pour le traitement en mémoire vive et SQLite pour le stockage permanent. En exploitant la directive PRIMARY KEY et l'indexation SQL, ce système garantit une recherche quasi-instantanée même sur de larges volumes de contacts.

typedef struct {
    int id;
    char nom[50];
    char tel[20];
} Contact;
CREATE TABLE contacts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    nom TEXT NOT NULL,
    tel TEXT UNIQUE NOT NULL
);
